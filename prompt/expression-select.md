목표:
EXPRESSION Step에서 MATCH Step에서 선택된 인물들을 기준으로
각 인물별 표정을 선택할 수 있는 UI를 구현한다.

기본 UI 구조:
- 상단: MATCH Step에서 선택된 인물 버튼 목록
  - 각 버튼은 인물 이미지 + 인물 번호(1~5)로 구성
  - 최대 5명까지 표시
- 하단: 표정 선택 UI 영역

상태 기본값:
- 초기 상태에서는 어떤 인물도 선택되지 않음
- 인물이 선택되지 않은 경우 하단 영역에 안내 문구를 표시한다.

인물 선택 시 동작:
- 인물 버튼을 클릭하면 해당 인물이 active 상태가 된다.
- active 인물이 있을 때 하단에 표정 선택 UI가 표시된다.

표정 선택 UI 요구사항:
- 이모티콘 기반 표정 버튼 10개를 나열한다.
- 이모티콘은 😄 😢 😡 등 다양한 희노애락을 표현한다.
- 표정 선택은 선택 사항(optional)이다.
- 표정 선택 UI 하단 또는 옆에 "초기화" 버튼을 표시한다.

표정 선택 동작:
- 이모티콘을 클릭하면:
  - active 인물에 해당 표정이 선택된다.
  - 선택된 표정 이모티콘이 해당 인물 버튼에 함께 표시된다.
- 이미 선택된 표정을 다시 클릭하면:
  - 선택 상태만 유지 (토글 여부는 선택)

초기화 버튼 동작:
- 초기화 버튼을 클릭하면:
  - active 인물의 표정 선택이 취소된다.
  - 인물 버튼에 표시된 이모티콘도 제거된다.

다중 인물 처리:
- 각 인물은 독립적으로 표정을 가질 수 있다.
- 예:
  - 인물 1 → 😄 선택
  - 인물 3 → 😡 선택
  - 인물 2, 4, 5 → 표정 없음
- 인물 간 표정 상태는 서로 영향을 주지 않는다.

UX 요구사항:
- active 인물 버튼은 시각적으로 구분 (border, highlight 등)
- 표정이 선택된 인물은 버튼 내에 이모티콘이 함께 표시됨
- 인물을 바꾸면 하단 표정 UI는 해당 인물의 상태를 반영한다.

---
이모티콘 선택하면 인물 원 안에 이모티콘이 들어가 있어 이모티콘이 잘 보이지 않는다. 인물 원 위에 이모티콘이 표시되도록 수정한다. (첨부 사진과 같이)
